Tables in HTML

Tables are used to represent real life table data

tr td th
used to display table row used to display table data used to display table header

Example:

<table>

<caption> Table Caption </caption>

<tr>

   <th> header 1 </th>

  <th> header 2 </th>

</tr>

<tr>

  <td> data 1 </td>

  <td> data 2 </td>

</tr>

</table>

Semantic table

thead to wrap table header
tbody to wrap table body
tfoot to wrap table footer

Colspan & Rowspan Attribute

used to create cells which spans over multioples rows and columns

rowspan = "" colspan = " "
