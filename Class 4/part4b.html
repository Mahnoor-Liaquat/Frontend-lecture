Forms in HTML

Forms are used to collect data from the user

Action Attribute

Action attribute is used to define what action needs to be peroformed when a form is submitted or where the form data should be sent

Example

< form action = "/action.js">

Form Element

Input

  • used to create multiple form controls.
  • there are multiple types of inputs that can be created using type attribute

Type Attribute

<input>

<input type = "text">

<input type = "password">

<input type = "number">

<input type = "time">

<input type = "color">

<input type = "range">

<input type = "date">

<input type = "radio">

<input type = "checkbox">

Placeholder attribute

<input type = "text" placeholder = "enter your name">

Label

label element represents a caption for an item in a user interface

<label for="name"> Enter your name: </label>

<input type = "text" placeholder = "enter your name" id="name">

Button

<button>Submit </button>

button type

  • submit
  • button
  • reset

Input Element

button using input

<input type = "submit"> value = "click me!"

Name Attribute

Name of the form control submitted with the form as part of a name / value pair

<input type = "text" placeholder = "enter your name" id="name" name= "username">
